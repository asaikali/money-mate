# Money Mate Agent

Spring AI MCP (Model Context Protocol) server for AI agent interaction.

## Description

This module provides an MCP server that exposes tools for AI agents (Goose, etc.) to interact with the Money Mate API. It uses Spring AI MCP Boot Starters for annotation-based server implementation.

## Technology Stack

- Spring Boot 3.5.8
- Spring AI (MCP Support)
- Java 25

## Status

This module is currently a placeholder skeleton. MCP implementation will be added in later phases.

## Running

From the money-mate-agent directory:

```bash
../mvnw spring-boot:run
```
